BTC/USDT Probabilistic K-Line Forecast Model

A Research Demonstration of Probabilistic Forecasting

Abstract

This project utilizes a deep learning model, enhanced with Monte Carlo methods, to generate probabilistic forecasts for the hourly (1h) K-line data of the BTC/USDT trading pair on the Binance exchange. By generating multiple (N) independent predictions, we can not only estimate the expected trend but also quantify the forecast uncertainty.
