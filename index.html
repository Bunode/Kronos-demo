Methodology Overview

This demonstration showcases the forecasting capabilities of Kronos, a foundation model pre-trained on the "language" of financial markets. The predictions are generated using the following process:

  • Model: The `Kronos-small` (25M parameters) model is used to autoregressively predict future K-line data.
  • Probabilistic Forecasting: We employ Monte Carlo sampling (N=10 paths) to generate a distribution of possible future price trajectories, not just a single point forecast. This allows us to quantify uncertainty and calculate probabilities.
  • Data: The model uses the last 360 hours (~15 days) of BTC/USDT 1h K-line data from Binance as context.
  • Automation: This page is updated automatically every hour by a script that fetches new data, runs the model, generates all assets, and pushes the changes to GitHub.

About The Kronos Project

Kronos is the first open-source foundation model for financial candlesticks (K-lines), trained on data from over 45 global exchanges. It is designed to serve as a unified model for diverse quantitative finance tasks.
